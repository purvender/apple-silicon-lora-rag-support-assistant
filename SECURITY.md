# Security

This project is an **educational lab** for local LLMs. It is not hardened production support software.

## Please do not

- Commit real customer data, API keys, or private policies to the repo or issues.
- Use fictional AcornDesk examples as legal advice for a real company.

## Report a security issue

If you find a security problem in the project itself (not your local setup), open a private security advisory on GitHub or email the maintainer listed in the repository profile.

Do not open public issues for sensitive vulnerability details.
